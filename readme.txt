Это работа Максимовой Анны Юрьевны

Код программы храниться в файле landscape.py
В данной программе используется библиотека PIL(т.е. Pillow), в частности модули Image, ImageDraw, ImageFont

Само изображение строиться за счет построения различных фигур с помощью метода polygon(), рисующего многоугольники по точкам, заданным последовательно, и закрашивающего их указанным цветом

Более детально информация о том, что делает тот или иной участок кода, я указала ввиде комментарий в коде программы

В конце используется метод truetype для добавления стороннего шрифта, поддерживающего русский язык
Метод text() добавляет надпись в изображение
